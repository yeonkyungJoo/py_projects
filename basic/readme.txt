1. 파이썬 기본
 - 단일 데이터(단일 타입)
  > 수치형 : 정수, 부동소수, 진법 등
  > 문자열(연속이지만, 단일로도 분류한다)
            :  표현, 사용 이유, 인덱싱, 슬라이싱, 포맷팅, 
               유니버셜 함수(주요 멤버 함수) -> 가장 중요
               전처리, 정규식 체크(데이터 수집 이후 등장)
  > boolean형 : 조건문 연결, 결론:False 상황만 기억
                False : 0, '', [], {}, (), None

 - 연속 데이터(시퀀스 타입)
  > 리스트   : [] -> js의 배열과 동일, 순서가 존재, 값 중복 OK
                    인덱스는 정방향/역방향(-1, -2, ...) 존재
  > 딕셔너리 : {} -> js의 객체와 동일, 순서 x, key:value 구조
                    키는 중복되면 X, 값 중복 OK
  > 튜플     : () -> 수정 불가, 값을 묶는다(관계없는 데이터들을 묶는다)
                    순서 존재, 정방향/역방향
  > 집합     : set() -> 중복 제거 후 리스트로 변환해서 처리
                       교집합, 합집합, 차집합 -> but, 느려서 pandas에서 해결

 - 반복문, 조건문, 제어문 등 -> 코드블럭 등장(:)
  > if, while, for, break, continue
 
 => 절차적 프로그램(동기식 작성)
 -------------------------------------------------------------------------
 - 함수
  > 함수 종류 : 내장/외장 함수, 사용자정의 함수, 
                람다표현(데이터분석 시 등장) -> 처리 속도나 메모리 점유가 더 좋다
 
 => 함수 지향적 프로그램(동기식 작성)
    -> 엔트리 포인트를 구성
 -------------------------------------------------------------------------
 - 클래스
  > 상속, 멤버, 생성자, 객체 생성
 
 - 모듈화 + 패키지
  > 모듈 만들기, 테스트 코드 배치, 모듈 가져오기, 패키지 사용
 
 - 예외처리
  > 풀 코드 형태 확인

 => 객체 지향적 프로그램
    -> GUI 모드 구성, 파이쎤 웹 구성시 모듈 부분으로 사용
