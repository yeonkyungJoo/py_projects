[[ python database program ]]

1. 용도
 - 시스템 통합에서 데이터베이스 역할
 - 데이터 수집에서 데이터를 저장하는 역할(저장소)
   적재(데이터를 읽어서 DataFrame으로 올리는 것, 전처리 후 저장하는 업무)

2. 라이브러리
 - 많다
 - pymysql을 사용하나, flask 자체에도 존재
 - 궁극의 모듈 sqlalchemy (이 모듈이 최종판)
   → pandas와 함께 주로 사용되고, 웹 서비스에서도 ORM 처리 방식으로 사용이 된다
   → 'sqlalchemy + 추가 라이브러리' 식으로 사용된다 : 브릿지 느낌
 - 풀링(Pooling)을 이용한 db와 접속 세션을 미리 확보하고 재사용하는 방식
